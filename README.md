# HoGent – Softwareontwerp 2  
## Eindevaluatie – Semester 1 (2025)

Dit is de eindevaluatie voor het vak Softwareontwerp 2, gegeven door Tommy Uytterhaegen.

## Keuzeopdrachten

Je kiest één van de volgende opdrachten:

- Ski-game (MonoGame)
- Kerstlijstje (MAUI)

Je mag beide opdrachten uitwerken, maar slechts één opdracht wordt gebruikt voor de evaluatie.

## Wat wordt verwacht

Voor de gekozen opdracht:

- Vul het [formulier](https://forms.cloud.microsoft/e/7mbRFJNGB7) in  
- Plaats op GitHub:
  - de volledige broncode (MonoGame of MAUI)
  - ontwerpdiagram
- Geef toegang tot de GitHub-repository aan tommy.uytterhaegen@hogent.be
- Dien de GitHub-link ook in via de opdracht op Chamilo

## Evaluatie

De focus van deze opdracht ligt op softwareontwerp, dit wil zeggen dat een zeer 'gelikte' user interface een goede oefening is en aangemoedigd wordt, maar het zal geen invloed hebben op de score. Forceer ook geen patronen, gebruik wat nuttig is. 

## Tips

Bekijk zeker de klassikaal gemaakte oefeningen (Surfing pikachu & Joke-app), eventueel ook de gedeelde code
